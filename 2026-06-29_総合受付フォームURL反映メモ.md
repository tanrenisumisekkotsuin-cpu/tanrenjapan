## script.js 差し替え方法

回答用URLは以下の `contact` に反映済み。

```js
const links = {
  contact: "https://forms.gle/SpMeMapQcsuRV6778",
};
```

管理画面URL、編集用URL、ログインURLは入れない。
必ず回答者が開ける公開用URLを入れる。
