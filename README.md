# TANREN JAPAN 公式案内ページ

TANREN の総合ご案内・相談窓口ページです。

## 公開URL

- Vercel: https://tanrenjapan.vercel.app

## ファイル構成

- `index.html` — ページ本体
- `style.css` — スタイル（案内ページ版）
- `script.js` — リンク設定（案内ページ版）
- `vercel.json` — Vercel 設定

## Vercel 連携

このリポジトリを Vercel に接続すると、`main` ブランチへの push ごとに自動デプロイされます。

- Framework Preset: **Other**
- Build Command: 空欄
- Output Directory: `.`
